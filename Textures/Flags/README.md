# Flags (European sovereign states)

PNG assets are named by **ISO 3166-1 alpha-2** country code (e.g. `de.png`, `gb.png`).

Downloaded from [flagcdn.com](https://flagcdn.com) (Wikimedia Commons–based, public domain). To refresh or change resolution, run:

```bash
python3 tools/download_europe_flags.py
```

Edit `EUROPE_ISO2` in that script if you need to add or remove territories.
